# Changelog

## [1.2.0] - Global Installation Support

### Added
- Global installation mode (`-g` / `-Global`) for user-level setup
- Installs to `~/.github-copilot/` (customizable via `GITHUB_COPILOT_HOME`)
- Works on both Unix/macOS and Windows

## [1.1.0] - Performance and Async Enhancements

### Added
- `instructions/async.md` - Async/concurrency patterns and best practices
- `instructions/monorepo.md` - Guidelines for monorepo workspace management
- `instructions/cicd.md` - Basic CI/CD patterns with GitHub Actions
- `agents/performance-optimizer.md` - Performance analysis and optimization agent
- `prompts/performance.md` - Performance optimization prompt
- Enhanced `templates/instructions-starter.md` with 3 worked examples

### Updated
- README.md - Updated contents tables with new files
- CONTRIBUTING.md - Minor clarifications
